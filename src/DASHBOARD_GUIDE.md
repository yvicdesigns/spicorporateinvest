# Dashboard User Guide: Managing Branch Cards

This guide explains how to update the images for the "Nos Pôles d'Excellence" (Our Centers of Excellence) section on the Homepage and Branches page.

## 1. Accessing the Image Library
1.  Log in to the **Dashboard**.
2.  Locate the module titled **Homepage Content** (Icon: Eye).
    *   *Note: Branch cards are considered global homepage content, so they are managed here rather than in individual pole sections.*
3.  Click to open the manager.

## 2. Uploading a New Branch Card Image
1.  **Add New Content** form will appear at the top.
2.  **Sections:** Check the box for **"Branch Card"**.
3.  **Associated Branch:** A new dropdown will appear. Select the branch this image belongs to (e.g., "SCI Renaissance").
    *   *Critical:* You must select the correct branch for the image to appear in the right card on the website.
4.  **Title:** Enter a descriptive title (e.g., "Renaissance Card Visual").
5.  **Image Upload:** Select your image file (JPG, PNG, WEBP).
6.  Click **Add Content Item**.

## 3. Editing Existing Images
1.  Scroll down to the **Existing Content** list.
2.  Use the tabs to filter by **"Branch Card"** to find your images easily.
3.  Locate the image you want to change.
4.  **To Change Metadata:** Click the **Edit (Pencil)** icon.
    *   You can change the "Associated Branch" here if an image is showing up for the wrong pole.
5.  **To Replace the Picture:** Click the **Image (Picture)** icon next to the edit button.
    *   Upload a new file to replace the visual while keeping the settings.

## 4. Troubleshooting
*   **Image not showing?** 
    *   Verify the image has the **"Branch Card"** section checked.
    *   Verify the **"Associated Branch"** tag matches the specific pole (e.g., "La Manne" for the agriculture card).
*   **Changes not reflected immediately?**
    *   Try refreshing the website page. The system uses a fast cache, but a hard refresh ensures you see the latest database version.
